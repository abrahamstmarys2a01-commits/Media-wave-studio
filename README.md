# Space Studio

Premium podcast studios designed for creators, businesses, and storytellers.

## Project Structure

This project contains the frontend React application for Space Studio.

- `frontend/`: Contains the Vite + React application.

## Quick Start

1. Navigate to the `frontend` directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
